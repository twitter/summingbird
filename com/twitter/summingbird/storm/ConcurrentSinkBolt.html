ConcurrentSinkBolt

class ConcurrentSinkBolt[StoreType <: Store[StoreType, (Key, BatchID), Value], Key, Value] extends SinkBolt[StoreType, Key, Value]

Extension of SinkBolt that allows concurrent writes from multiIncrement. WARNING: Only use this with ConcurrentMutableStores. The type bounds with type erasure (in the CompleteBuilder pattern match) don't allow the type system to enforce this constraint.
